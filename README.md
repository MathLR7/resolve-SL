# resolve-SL
A program developed to solve systems of linear equations. This project was created as a practical assignment for the Introduction to Programming course at UFES (Federal University of Espírito Santo).  

To run the program correctly, you need to pass the size of the linear system's square matrix as a command-line argument. e.g., ".\a.exe 5"
